# ⚡ Aithentik ALPHA-LOOP v2.0

Dein persönliches kybernetisches Nervensystem für Marktdominanz und Mustererkennung.

## Architektur
- **Backend**: Python-Daemon mit PAL (Pattern Abstraction Layer) und Society of Mind.
- **Frontend**: High-Performance React Dashboard (Vite + Tailwind + Recharts).
- **Persistence**: Qdrant Vector Database (`myzel` Collection).
- **Intelligence**: DeepSeek R1 via Ollama.

## Schnellstart
1. **Infra**: `docker compose up -d`
2. **Backend**: 
   ```bash
   cd backend
   pip install -r requirements.txt
   python alpha_loop_daemon_v2.py
   ```
3. **Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

## Key Features
- **PAL Logic**: Extrahiert mechanische DNA aus jedem Input.
- **Cash Score**: Automatische Bewertung von Mustern nach Dominanz und Anomalie.
- **Myzel Search**: Findet Isomorphien zwischen Trading, Kunst und Tech.
- **Auto-Scan**: Überwacht HackerNews, arXiv und Krypto-Feeds autonom.
